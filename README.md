# Statistical-Dissertations
A collection of statistical dissertations exploring various methodologies and academic applications, implemented using R and Jamovi.

| Project Name | Description |
| :--- | :--- |
| **[Factors Affecting the Habitual Consumption](https://github.com/lacson06/Statistical-Dissertations/blob/main/Factors%20Affecting%20the%20Habitual%20Consumption.pdf)** | An analysis of a research paper from World Citi Colleges students examining the factors influencing habitual consumption. It utilizes a comprehensive suite of statistical methodologies, including descriptive statistics, independent t-tests, correlation analysis, ANOVA, and linear regression. |
